# Guía de Inicio Rápido

## 1. Instalar Bun

Ejecuta este comando en tu terminal (PowerShell):

```powershell
powershell -c "irm bun.sh/install.ps1 | iex"
```

## 2. Clonar el Proyecto

```bash
git clone <tu-repositorio-url>
cd Nose
```

## 3. Instalar y Ejecutar

```bash
bun install
bun run dev
```
